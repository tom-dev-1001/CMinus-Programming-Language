Phase 1:
1. Get enums and struct definitions ✅
2. Resolve struct members ✅
3. Get function names, return types and parameters ✅
4. Get global vars ✅
5. Add function names and global names to symbol table ✅

Phase 2:
1. Resolve expression types:

    1. assign type id for ast nodes
    2. check operator compatibility, array indexing types, pointer dereference

2. validate assignments: 

    both sides have types, compatible, not const

3. validate functions: 

    parameters, local scopes, validate return, missing returns

4. control flow checks

    if condition is bool
    while condition is bool
    for valid
    break, continue

5. function calls

    function exists, return type is valid, parameters match

6. identifier resolution

    resolve references to:
        globals
        locals
        parameters
    assign symbol and type ids

7. final annotation pass

    every expression node has a type_id
    every identifier has a symbol_id
